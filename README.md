# Bank System Project

A simple **Python bank system** package that demonstrates core banking operations such as account creation, deposits, withdrawals, and balance checks.  
Built with a clean package structure and test support.

---

---

## ✅ Features

- Create and manage bank accounts
- Deposit and withdraw funds
- Check account balance and basic validations
- Unit tests included under `tests/`

> Note: Feature list may vary depending on your implementation inside `bank_system/`.

---

## 🛠️ Requirements

- Python 3.10+ (recommended)
- `pip` installed
